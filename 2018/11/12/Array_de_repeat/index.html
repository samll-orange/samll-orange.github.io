数组去重

发表于 2018-11-12 | 分类于 JavaScript

在原型链上进行操作

Array.prototype

  1. 思路：
  • 利用对象属性名的不重复性，以及属性名不存在时值为：undefined，
  • 通过for循环进行筛选，把不重复的属性名放入到新的数组里面去。

具体的分析如下：

  • for循环：对于目标数组，把数组的每一位拿出来判断数值是否重复，获得目标数组的长度作为循环的长度。
  • 把数组的每一位放到对象里面作为属性名，挨个访问对象的属性名判断是否有该属性名。
  • 判断数组当前位的值放到对象里面，看看是否已经存在值，如果没有的话，就把一个值放进去。否则，就不放值进去。
  • 拓展成一个高级的写法：有个问题–无法给0进行去重
